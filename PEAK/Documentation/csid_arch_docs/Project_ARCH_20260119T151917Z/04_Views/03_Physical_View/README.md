# Physical View

**Purpose:** Describe the physical realization (hardware/software deployment, network, etc.).

## Include
- Physical nodes (devices, ECUs, servers, sensors, actuators)
- Deployment / allocation: logical → physical
- Physical interfaces (buses, networks, connectors)
- Constraints (mass/power/latency/environment)

## Prompts
- What are the candidate physical platforms?
- What constraints dominate (power, weight, connectivity, cost)?
- What are the key deployment scenarios?

## “Good looks like”
- Clear mapping from logical to physical
- Constraints stated with units/limits where possible
