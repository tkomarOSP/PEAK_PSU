# Logical View

**Purpose:** Describe the logical decomposition and interactions that realize the functions.

## Include
- Logical components / subsystems
- Logical interfaces and exchanges
- Allocation: functions → logical components
- Key interaction patterns (message/event/data flow)

## Prompts
- What are the main logical responsibilities?
- What interactions must exist regardless of physical tech choice?
- Where are the major performance, safety, security drivers?

## “Good looks like”
- Coherent responsibilities (low coupling, high cohesion)
- Clear interface definitions
